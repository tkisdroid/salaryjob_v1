### Pattern reference

All inline warning `<p>` elements use exactly the same class as Step 3 (original line 592-596):
```tsx
{form.hourlyPay < MINIMUM_WAGE && form.hourlyPay > 0 && (
  <p className="text-[10px] font-bold text-destructive">
    최저시급({formatMoney(MINIMUM_WAGE)}) 이상이어야 합니다
  </p>
)}
```
Progressive disclosure rule: warning only appears after user has started typing (`> 0` / `!== ""`), not in the blank initial state.
