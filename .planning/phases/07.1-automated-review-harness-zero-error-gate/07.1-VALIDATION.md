## Sampling Rate

- **After every task commit:** Run `vitest run` + any touched Playwright spec.
- **After every plan wave:** Run `npm run review:quality` (tsc + eslint + vitest + e2e smoke).
- **Before `/gsd:verify-work`:** Full `npm run review` must exit 0 with `production_ready: true` in `07.1-REVIEW.md`.
- **Max feedback latency:** 30 seconds for Wave 1 tasks; 30 minutes for full sweep.
